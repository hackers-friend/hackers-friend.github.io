Important Formula :

Speed of the boat in still water:

  = (down stream speed  +  upstream speed ) / 2

   Bs = ( Ds  + Us ) / 2

Speed of current:

= (down stream speed  -  upstream speed ) / 2

Cs = ( Ds  - Us ) / 2

Down stream speed :

down stream speed = speed of the boat + speed of the current

Ds = Bs + Cs

Upstream speed :

Upstream speed = speed of the boat - speed of the current
