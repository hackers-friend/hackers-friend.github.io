This type of questions are based on the concept of time ,distance and speed.

Conversion between m/s to km/hr :

p m/s =( p * 18/5) km/hr

comversion between km/hr to m/s :

p km/hr = ( p * 5/18) m/s

Concept of Speed :

When one object is stationary nd another is still then only speed of moving object is considered as S1 = x (let) (S2 = 0) 

  • When both object s are moving in same direction :
    • In this case relative speed is difference between speed of both objects that means realtive speed = S1 - S2 
  • When both object are moving in opposite direction :
    • In this case relative speed is sum of both speed of both object means relative speed = S1+ S2

Concept of Distance :

Distance is always added to both object with irrespective of direction but in some exception are there, when any vertical object is taken then distance of that object is considered as zero.

ex: Length of pole, person, tree is zero.

Concept of Time :

Time is always  constant ( The time which is taken by object 1st to cross 2nd)

Some Important Point :

  • When a moving train crosses a standing person i.e. : S1 = L1 / T
  • When a moving train crosses a moving persons in the same direction : S1 - S2 = L1 / T  and L2 = 0
  • When a moving train crosses a moving persons in the opposite direction : S1 + S2 = L1 / T
  • When a moving train crosses a plateform : S1 = L1 + L2 / T , S2 = 0
  • When a moving train crosses another stationary train : S1 = L1 + L2 / T , S2 = 0
  • When a moving train crosses another moving train along same direction : S1 - S2 = L1 + L2 / T
  • When a moving train crosses another moving train coming from oppsosite direction : S1 + S2 = L1 + L2 / T
  • When a moving train crosses a person sitting in another train : S1 ± S2 = L1 / T, L2 = 0 , S1 + S2 (according to direction)
