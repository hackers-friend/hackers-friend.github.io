Important Formulae :

 

  • (a + b)(a - b) = (a² - b²)
  • (a + b)² = (a² + b² + 2ab)
  • (a - b)² = (a² + b² - 2ab)
  • (a + b + c)² = a² + b² + c² + 2(ab + bc + ca)
  • (a³ + b³) = (a + b)(a² - ab + b²)
  • (a³ - b³)  = (a - b)(a² + ab + b²)
  • (a³ + b³ + c³ - 3abc) = (a + b + c)(a² + b² + c² - ab - bc - ac)
  • When a + b + c = 0, then a³ + b³ + c³ = 3abc
