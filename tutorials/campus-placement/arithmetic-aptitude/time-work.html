Time and Work

  • Learn

Relation between Work and Days :

  • If P can do a piece of work in x days, then P's one day's work = 1/x.

Relation between Days and work :

  • If P's one day's work = 1/x, then P can finish the work in x days.

Ratio:

If P is thrice than good workman as Q, then

ratio of work done by P and Q  = 3 : 1.

ratio of times taken by P and Q to finish a work = 1 : 3.

Important Formulas :

  • If A can do a piece of work in X days and B can do the same work in Y days , then both of them working together will do the same work in :  (XY / X + Y) days.
  • A,B,C while working alone can complete a work X,Y and Z days respectively then they will together complete the work in : (XYZ / XY + YZ + ZX).
