Ration and Proportion

Ratio :

a : b = a/b = a÷b

Compounded Ratio :

(a : b)(c : d)(e : f) → (ace : bdf)

Duplicate Ratio:

a : b = a² : b²

Triplicate Ratio :

a : b = a³ : b³

Comparison of Ratios:

Let  a : b and c : d be two ratios, then a : b > c : d if ad > bc 

i.e a / b > c / d if ad > bc

Similarly ,

a : b < c : d if ad < bc

a : b = c : d if ad = bc

Important Point :

If A : B = x : y and B : C = p : q then 

  • A : C = x × p : y × q
  • A : B : C = (x : y) × p : y × q

If A : B = x : y, B : C = p : q and C : D = m : n, then

  • A : D = x × p × m : y × q × n
  • A : B : C = xpm : ypm : yqm : yqn

Proportion :

Third Proportion :

If  a : b :: b : c, then c is called the 3rd proportional to a and b. c will be calculated as below a : b :: c : c or a : b = b : c

⇒ a × c =b × b 

∴ c = b² / a

Fourth Proportion:

If  a : b :: c : d, then d is called the 4th proportional to a , b and c. d will be calculated as below a : b :: c : d or a : b = c : d

⇒ a × d =b × c 

∴ d = bc / a

Mean Proportion:

If a and b then mean proportion is √(ab)

If mean proportion between a and b is A then:

a : A :: A : b

⇒ A² = ab

or

a × b = A × A

∴ A = √(ab)
