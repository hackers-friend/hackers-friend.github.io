In this tutorial we'll learn about how to store models and restore them when needed.

What is TensorFlow model ?

Once we train a neural network, what we want to do is, save it for future use and deploying to production. There comes, models. Tensorflow model primarily contains the network design or graph and values of the network parameters that we have trained.

Tensorflow model has mainly two files:

  1. Meta graph
  2. Checkpoint Files.

Meta graph is a protocol buffer. It saves the complete Tensorflow graph; i.e. all variables, operations, collections etc. This file has .meta extension.

Checkpoint File is binary file. It contains all the values of the weights, biases, gradients and all the other variables saved. This file has an extension .ckpt.

Update: from version 0.11, instead of 1 .ckpt file, we have two files.

  1. mymodel.data-00000-of-00001
  2. mymodel.index

our training variables are in .data files. Along with these files, we have a file named checkpoint which only keeps a record of latest checkpoint files saved.

Let's come back to point now.

How to save a TensorFlow Model ?

Suppose we are training a CNN for Image classification. As a standard practice, we'll keep track of loss and accuracy numbers. Once we find that the network has converged, we can stop the training manually or we will run the training for fixed number of epochs. After the training is done, we want to save all the variables and network graph to a file for future use. So, in Tensorflow, to do that, we shall be creating an instance of tf.train.Saver() class.

saver = tf.train.Saver()

Note that Tensorflow variables are only alive inside a session.

So, you have to save the model inside a session by calling save method on saver object which we just created.

 

saver.save(sess, 'hackersfriend-model')

Here, sess is the session object, while ‘hackersfriend-model’ is the name we want to give to our model. Here is full code:

import tensorflow as tf

w1 = tf.Variable(tf.random_normal(shape=[2]), name='w1')
w2 = tf.Variable(tf.random_normal(shape=[5]), name='w2')
saver = tf.train.Saver()
# Creating a session with name sess
sess = tf.Session()

sess.run(tf.global_variables_initializer())

# Saving the model with name hackersfriend-model
saver.save(sess, 'hackersfriend-model')

# This will generate following files
# hackersfriend-model.data-00000-of-00001
# hackersfriend-model.index
# hackersfriend.meta
# checkpoint

 

Now suppose, we are saving the model after 1000 iterations. Then, we shall call save by passing the step count:

saver.save(sess, 'hackersfriend-model',global_step=1000)

This will just append ‘-1000’ to the model name and following files will be generated:

 

  1. hackersfriend-model-1000.index
  2. hackersfriend-model-1000.meta
  3. hackersfriend-1000.data-00000-of-00001
  4. checkpoint

In this particular example, while training, we are saving our model after every 1000 iterations, so .meta file is created the first time(on 1000th iteration) and we don’t need to recreate the .meta file each time(so, we don’t save the .meta file at 2000, 3000.. or any other iteration). Since, graph will not change, we'll only save the model for further iterations.We don't need to meta graph again and again. So, in code, we'll do it this way.

saver.save(sess, 'hackersfriend-model', global_step=step,write_meta_graph=False)

Suppose, we want to keep only 5 latest models and want to save 1 model after every 2 hours. To acheive this, we can  use max_to_keep and keep_checkpoint_every_n_hours during training. Here is how to code it.

saver = tf.train.Saver(max_to_keep=5, keep_checkpoint_every_n_hours=2)

Now, let's get more control over what we store. What if, we don’t want to save all the variables and just some of them.  When we dont specify, anything in the tf.train.Saver(), it saves all variables.

While creating the tf.train.Saver instance we can specify what we want to save. We have to pass it as list or dictionary of variables. Let's code it.

import tensorflow as tf
w1 = tf.Variable(tf.random_normal(shape=[2]), name='w1')
w2 = tf.Variable(tf.random_normal(shape=[5]), name='w2')
saver = tf.train.Saver([w1,w2])
sess = tf.Session()
sess.run(tf.global_variables_initializer())
saver.save(sess, 'hackersfriend-model', global_step=1000)

 

That was all about how to save a model. Now, let's understand how to re-use any saved model.

How to import a pre-trained model in TensorFlow ?

To use any-pretrained model in Tensor we have to first create network and then load varaibles. For better understanding, we have broken it down into steps.

  1. Create Network   

 To create Network, we have 2 ways. One is to re-create each and every layer manually as the original model. And 2nd is, to  use .meta file file and recreate the network using tf.train.import() function. Here is how code it.

saver = tf.train.import_meta_graph('hackersfriend-model-1000.meta')

What happens behind scences when you do it is, import_meta_graph appends the network defined in .meta file to the current graph. 

Now, this way, we have got the graph or network, which we need. But, it still needs, value of prameters. Let's do that as well. 

  1. Load the parameters

To load parameters of previously stored model, in our current model, we can call restore on instance of tf.train.Saver() class. Let's code it. 

with tf.Session() as sess:    
    saver = tf.train.import_meta_graph('hackersfriend-model-1000.meta')
    saver.restore(sess,tf.train.latest_checkpoint('./'))
    print(sess.run('w1:0'))  #This will work and print the saved value of w1. because Model has been restored.

Now, that we have restore our model. Let's start working with it.

How to Work with restored models in TensorFlow ?

Till now, we have learned about what is a TensorFlow Model, how to save a model and import it when needed. Now, let's get a practicle guide to how to use any model after resotoring it.

Here, we will resotre a saved model and use it for further training, prediction, fine tuning etc. 

Whenever we create a TensorFlow Model, we feed training data into it, and some other stuffs like, earning rate, global step etc. We feed all these data using placeholders.

Note: Whenever we save the network, values of placeholders are not saved. Let's code a network using placeholders and store it. 

import tensorflow as tf

# Let's create feed input, i.e. feed_dict and placeholders

w1 = tf.placeholder("float", name="w1")
w2 = tf.placeholder("float", name="w2")
b1= tf.Variable(2.0,name="bias")
feed_dict ={w1:4,w2:8}

#Define a test operation that we will restore
w3 = tf.add(w1,w2)
w4 = tf.multiply(w3,b1,name="op_to_restore")
sess = tf.Session()
sess.run(tf.global_variables_initializer())

#Create a saver object which will save all the variables
saver = tf.train.Saver()

#Run the operation with feeding input
print sess.run(w4,feed_dict)
#Prints 24 which is sum of (w1+w2)*b1 

#Now, save the graph
saver.save(sess, 'hackersfriend-model',global_step=1000)

 

We have saved the network. Now, let's restore it.

Here, when we are trying to restore it, we not only have to restore the graph and weights, but also prepare a new feed_dict that will feed the new training data to the network. 

We can use graph.get_tensor_by_name() method to get reference to saved operations and placeholder variables.

#Accessing saved variable/Tensor/placeholders 
w1 = graph.get_tensor_by_name("w1:0")

#Acessing saved operation
op_to_restore = graph.get_tensor_by_name("op_to_restore:0")

 

Now, suppose, if we want to run run the same network with different data, we can do that too. We just have to pass the new data via feed_dict to the network.

import tensorflow as tf

sess=tf.Session()    

# we need to load meta graph and restore weights first.
saver = tf.train.import_meta_graph('hackersfriend-model-1000.meta')
saver.restore(sess,tf.train.latest_checkpoint('./'))


# Then, we will access and create placeholders variables and create feed-dict to feed new data

graph = tf.get_default_graph()
w1 = graph.get_tensor_by_name("w1:0")
w2 = graph.get_tensor_by_name("w2:0")
feed_dict ={w1:13.0,w2:17.0}

#Now, access the op that we want to run. 
op_to_restore = graph.get_tensor_by_name("op_to_restore:0")

print sess.run(op_to_restore,feed_dict)

 

we can use graph.get_tensor_by_name() method to restore part of the old graph and add-on to that for fine-tuning.
