TensorFlow - Introduction & Installation

  • Learn
  • Practice

What is TensorFlow ?

TensorFlow is an open source machine learning framework. It is used for implementing machine learning and deep learning applications. Google team created TensorFlow to develop and research on fascinating ideas on artificial intelligence. TensorFlow is designed in Python programming language.

It combines the computational algebra of optimization techniques for easy calculation of many mathematical expressions.

The official website of TensorFlow is  www.tensorflow.org

Before proceeding with this tutorial, we expect you to have a basic knowledge of Python programming language. Knowledge of artificial intelligence concepts will be a plus point.

What's special about TensorFlow ?

following important features of TensorFlow −

  • It includes a feature of that defines, optimizes and calculates mathematical expressions easily with the help of multi-dimensional arrays called tensors.

  • It includes a programming support of deep neural networks and machine learning techniques.

  • It includes a high scalable feature of computation with various data sets.

  • TensorFlow uses GPU computing, automating management. It also includes a unique feature of optimization of same memory and the data used.

TensorFlow is well-documented and includes plenty of machine learning libraries. It offers a few important functionalities and methods for the same.

TensorFlow is also called a “Google” product. It includes a variety of machine learning and deep learning algorithms. TensorFlow can train and run deep neural networks for handwritten digit classification, image recognition, word embedding and creation of various sequence models.

Installation

To install TensorFlow, you must have “Python 3.4+ ” installed in your system. 

Step 1 − Verify the python version being installed.

               type python in commanline and hit enter

C:\users\HackersFriend>python
Python 3.6.4 (v3.6.4:d48eceb, Dec 19 2017, 06:04:45) [MSC v.1900 32 bit (Intel)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>>

Step 2 − You can pick up any mechanism to install TensorFlow in the system. Though “pip” and “Anaconda” are easier once. Pip is a command used for executing and installing modules in Python.

Before we install TensorFlow, Let's install Anaconda framework in our system.

go to https://www.anaconda.com/ and download latest version and install it.

After successful installation, we are ready to move ahead.

Step 3 − Use pip to install “Tensorflow” in the system.

pip install tensorflow

after that

pip install tensorflow-gpu

 

After successful installation, let's check the sample program execution of TensorFlow.

as always, let's print “Hello World” in TensorFlow.

The code for Hello world is following:

>> activate tensorflow
>> python (activating python shell)
>> import tensorflow as tf
>> hello = tf.constant(‘Hello, Tensorflow!’)
>> sess = tf.Session()
>> print(sess.run(hello))

Congratulations. You printed Hello world using TensorFlow. Let's get started!
