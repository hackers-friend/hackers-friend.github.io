Logical Operators

  • Learn
  • Practice

Logical operators take boolean values as input and return boolean values as output. 
Note: In C,C++ any non-zero number is treated as true and 0 as false but this doesn't hold for Java.

Following table describes logical operators and their usage:

 

Symbol

Operation

Usage

Explanation

&&

logical AND

x && y

Returns true if both x and y are true else returns false.

||

logical OR

x || y

Returns false if neither x nor y is true else returns true

!

logical NOT

! x

Unary operator. Returns true if x is false else returns false.
