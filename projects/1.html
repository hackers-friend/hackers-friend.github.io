Responsive frontend developement Front end

Full description:

Goal of this Project is to develop a responsive frontend of one of our client. The web pages should look and work properly on all devices i.e. on mobile devices, tablets and desktops. You'll be given full details for color schema to use during design and other data which should show in front end. You'll also be given with the name of input fields you are required to use in forms. Backend accepts GET, and POST requests. Forms supposed to use only these methods.

Skills required: html, css, javascript, jquery, bootstrap 4, other front end libraries

Payment: Rs 1000 + Interview for job offers with client company to top 5 winners
